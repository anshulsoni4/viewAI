## Run Locally

Clone the project

```bash
  git clone https://github.com/anshulsoni4/viewAI
```

Go to the project directory

```bash
  cd view-ai project
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
